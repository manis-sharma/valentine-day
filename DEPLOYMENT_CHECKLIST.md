## ğŸ› Troubleshooting

If something goes wrong:

### Site Not Loading
1. Check GitHub Pages is enabled
2. Wait 5-10 minutes for deployment
3. Clear browser cache (Ctrl+Shift+R)
4. Check repository is Public
5. Verify correct branch selected

### Images Not Showing
1. Verify image URLs are correct
2. Check images are publicly accessible
3. Test URLs in new browser tab
4. Ensure HTTPS links (not HTTP)
5. Check file extensions (.jpg, .png, etc.)

### Animations Not Working
1. Test in different browser
2. Check JavaScript console for errors
3. Verify no syntax errors in code
4. Test on desktop first, then mobile

### Mobile Issues
1. Test responsive design
2. Check viewport meta tag
3. Verify touch events work
4. Test on actual device, not just emulator
